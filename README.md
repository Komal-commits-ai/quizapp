# 🧠 QuizApp – Full Stack MERN Quiz Platform

## 🚀 Project Overview

**QuizApp** is a full-stack web application built using the **MERN Stack (MongoDB, Express, React, Node.js)** that allows users to create quizzes, attempt them, and track their performance.

The platform is designed for **educational systems, practice exams, and online assessment tools**, providing a smooth and interactive quiz experience with a responsive modern interface.

This project demonstrates **end-to-end full stack development**, including frontend UI, backend APIs, database integration, and cloud deployment.

## 🏗 Application Architecture

The project follows a **full-stack MERN architecture**.

Frontend (React)
Handles UI, quiz interface, routing, and user interactions.

Backend (Node + Express)
Provides REST APIs for quizzes, attempts, and data management.

Database (MongoDB Atlas)
Stores quiz data, questions, attempts, and results.

---

## 🧩 Core Modules

### Quiz Creation

Users or admins can create quizzes by adding questions and answer options.

### Quiz Attempt Interface

Interactive quiz interface allowing users to answer questions sequentially.

### Result & Score Tracking

Scores are calculated and displayed after quiz completion.

### Attempt History

Users can view previously attempted quizzes and analyze performance.

### Data Management

Backend APIs manage quizzes, questions, and attempt records.

## ⚡ Local Installation

### Clone the repository

git clone https://github.com/Komal-commits-ai/quizapp.git

---

### Backend Setup

cd server
npm install
npm start

Server runs on:
http://localhost:5000

---

### Frontend Setup

cd src
npm install
npm start

Frontend runs on:
http://localhost:3000

Make sure API calls point to the backend server during development.
